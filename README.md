# Ngoma Charts

**Live site → [ngomacharts.netlify.app](https://ngomacharts.netlify.app/)**

A unified music chart for Kenya. Instead of checking Spotify, Apple Music, Audiomack, Boomplay, YouTube, and Shazam separately, Ngoma Charts aggregates rankings across all six platforms into one combined Ngoma Top 50 list.

## Stack

- Vite + React + TypeScript
- Tailwind CSS
- Framer Motion
- Deployed on Netlify

## Platforms Tracked

| Platform | Data Type |
|---|---|
| Spotify | Streaming Charts |
| Apple Music | Streaming Charts |
| Audiomack | Streaming + Downloads |
| Boomplay | Streaming Charts |
| YouTube | Music Video Views |
| Shazam | Discovery & Shazams |
